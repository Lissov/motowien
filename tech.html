Home Routes Technologie

Diese Webseite ist auf pures statisches HTML und CSS gemacht. Ist minimalistisch klein und hoffentlich nicht all zu hässlich.

Natürlich folge ich DRY prinzip (Don't repeat yourself - wiederhole sich nicht). Deshalb habe ich Menu, Footer u.s.w. als separate HTML-files gemacht, die ich mit Powershell Skript in alle Seiten einbringe. Auf diese Weise brauche ich nur ein enfaches Powershell Kommand vor Deployment, dann bleibt die Webseite vollig statisch und braucht gar keine externe Tools oder Code. Für deployment kann ich nur ein Paar Dateien auf Server hochladen.

Ich habe Visual Studio Code benutzt.

Als Hosting nutze ich Azure, was vollig unnötig für so ein simples Projekt ist :)
